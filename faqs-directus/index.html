FAQS Directus

How to see / edit Raw Value?

To edit Raw Value you need to:

  • Open an item page
  • Click on the field name you want to see/ edit the value
  • Click on Raw Value

How to inspect / debug my Directus instance?

To run directus with inspector enabled you need to:

  • Run the following command inside your directus instance folder: 
    node --inspect node_modules/.bin/directus start
  • Attach debugger via VSCode or another debugger client, like Chrome
